## How To Use

From your command line:

#### # Clone this repository

```bash
git clone https://github.com/Tawakkalna/react-native.git
```

#### # Go into the frontend directory

```bash
cd react-native/TawakklnaA
```

#### # Install the dependencies

```bash
yarn install
```

#### # Run the app

```bash
npm start
```
